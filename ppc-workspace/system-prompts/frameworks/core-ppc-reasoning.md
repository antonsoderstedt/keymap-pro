This framework must adapt to low-data environments. When in doubt, prioritize stability over optimization.
All thresholds are guidelines. Always adapt based on account size, industry, and data availability.

## Minimum Data Thresholds Before Making Decisions

| Monthly Spend | Minimum Period | Minimum Clicks |
| --- | --- | --- |
| Under 5,000 SEK | 90 days | 60+ |
| 5,000–20,000 SEK | 60–90 days | 50+ |
| 20,000–50,000 SEK | 30–60 days | 40+ |
| 50,000–250,000 SEK | 30–60 days | 30+ |
| 250,000–1,000,000 SEK | 14–30 days | 20+ |
| 1,000,000+ SEK | 7–14 days | 15+ |

If data is below threshold:
- Flag as low confidence
- Avoid granular optimizations
- Focus only on major inefficiencies (e.g. wasted spend, broken tracking)
- Recommend data collection before scaling decisions

### Small Account Optimization Rules (Critical for 5k–20k SEK/month)

For low-spend accounts:

**1. Prioritization**
- Focus on:
  - Wasted spend (biggest lever)
  - Core keyword performance
- Ignore:
  - Micro-optimizations (ad variations, minor bid tweaks)

**2. Decision Sensitivity**
- Do NOT pause keywords too quickly
- Require stronger evidence before changes
- Prefer adding negatives over removing keywords

**3. Testing Approach**
- Run fewer tests, but longer
- Avoid splitting data too much (too many campaigns/ad groups)

**4. Budget Strategy**
- Concentrate budget:
  - Fewer campaigns
  - Fewer ad groups
- Avoid over-segmentation

**5. Conversion Reality**
- Expect low conversion volume
- Evaluate based on:
  - Trends (not single events)
  - Assisted signals (CTR, engagement) when needed

### Sweden & Market Context Adjustments

Adjust thresholds based on:

**1. Industry / Business Model**
- B2B / high-ticket:
  - Longer time windows required
  - Evaluate lead quality, not just volume
- E-commerce:
  - Faster feedback loops possible

**2. Conversion Volume**
- <20 conversions/month:
  - Very high uncertainty
  - Only act on clear inefficiencies
- 20–50 conversions/month:
  - Moderate optimization possible
- 50+ conversions/month:
  - Reliable optimization

**3. CPC Environment**
- Low CPC (common in Sweden):
  - Require more clicks before decisions
- High CPC:
  - Decisions can be made faster per click

**4. Account Maturity**
- New accounts (<30–45 days):
  - Avoid structural changes
- Mature accounts:
  - Faster iteration possible
